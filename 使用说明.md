# Claude Code Windows 一键安装工具

## 简介

本工具帮助 Windows 新手用户一键配置 Claude Code 环境。

## 系统要求

- Windows 10 或 Windows 11
- 网络连接

## 使用方法

### 方法一：双击运行（推荐）

1. 双击 `一键安装ClaudeCode.bat`
2. 按提示操作即可

### 方法二：PowerShell 运行

```powershell
powershell -ExecutionPolicy Bypass -File install-claude-code.ps1
```

## 安装内容

脚本会自动完成以下操作：

1. ✅ 检测系统环境
2. ✅ 安装 Node.js（如未安装）
3. ✅ 安装 Claude Code
4. ✅ 创建工作目录
5. ✅ 创建桌面快捷方式

## 首次使用

安装完成后，需要配置 API Key：

1. 双击桌面的 "Claude Code" 快捷方式
2. 按提示登录 Anthropic 账号
3. 开始使用！

## 常见问题

**Q: 安装失败怎么办？**
A: 尝试以管理员身份运行脚本

**Q: 如何更新 Claude Code？**
A: 运行 `npm update -g @anthropic-ai/claude-code`
